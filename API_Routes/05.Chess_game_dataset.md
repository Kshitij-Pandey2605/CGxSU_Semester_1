# Match Management & CRUD Routes

| Method       | Endpoint                         | Description                |
| :----------- | :------------------------------- | :------------------------- |
| **`GET`**    | `/api/v1/matches`                | Fetch all chess matches    |
| **`GET`**    | `/api/v1/matches/:id`            | Fetch single match details |
| **`POST`**   | `/api/v1/matches`                | Create new chess match     |
| **`PUT`**    | `/api/v1/matches/:id`            | Replace full match record  |
| **`PATCH`**  | `/api/v1/matches/:id`            | Update match fields        |
| **`DELETE`** | `/api/v1/matches/:id`            | Delete chess match         |
| **`GET`**    | `/api/v1/matches/:id/moves`      | Fetch match moves          |
| **`GET`**    | `/api/v1/matches/:id/pgn`        | Fetch PGN notation         |
| **`GET`**    | `/api/v1/matches/:id/summary`    | Match summary              |
| **`GET`**    | `/api/v1/matches/:id/statistics` | Match statistics           |
| **`GET`**    | `/api/v1/matches/latest`         | Latest matches             |
| **`GET`**    | `/api/v1/matches/trending`       | Trending matches           |
| **`GET`**    | `/api/v1/matches/random`         | Random chess match         |
| **`PATCH`**  | `/api/v1/matches/:id/archive`    | Archive match              |
| **`PATCH`**  | `/api/v1/matches/:id/restore`    | Restore archived match     |
